# Donation-attack-into-VRC
Donation attack into VRC(Daiv)는 투네이션과 트윕의 알림을 받아서 OSC를 통해 VRC에 파라메터값을 보내주는 프로그램입니다.

## 사전설정
이 프로그램은 VRC 파라메터값을 전송해주는 프로그램이므로 아바타를 업로드할때 파라메터값에 대한 애니메이션을 미리 등록해두셔야됩니다.
